# iNotebook Backend

This is the backend of the iNotebook application using Mongodb as a database

## Features

- Sign up to create an account
- Add multiple notes
- Edit your notes
- Delete your notes
- Add a new note
- Cann't login with wrong password
- Provide password security using salt 
- Use of Api
- Use of middleware

## Package Used:

- Express
- Mongoose
- Nodemon
- Express-validator
- Bcryptjs
- Jsonwebtoken
